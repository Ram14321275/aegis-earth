# Economic Topology

Aegis Earth treats global economics, supply chains, and resource flow as a deterministic, mathematically provable intelligence layer.

1. **Market Signal Engine**: Ingests volatility and evaluates staleness to compute a deterministic confidence score (no opaque AI guessing).
2. **Resource Allocation**: Resolves scarcity using mathematically weighted criticality formulas distributed across affected infrastructure.
3. **Disruption Propagation**: Projects economic cascading failures purely on root severity multiplied by dependency graphs.
