# Aegis Earth Frontend

Official Sprint 1 frontend workspace for Aegis Earth.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui-style local components
- Framer Motion
- Leaflet

## Structure

```text
frontend/src/
  assets/
  components/
  features/
  hooks/
  pages/
  routes/
  services/
  store/
  types/
```

## Commands

```bash
npm install
npm run build
npm run dev
```
