# Aegis Earth Frontend

Official Sprint 1 frontend workspace for Aegis Earth.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui-style local components
- Framer Motion
- Leaflet
- React Router

## Structure

```text
frontend/src/
  assets/
  components/
  constants/
  layouts/
  features/
  hooks/
  pages/
    Home/
    Dashboard/
    Analysis/
    NotFound/
  routes/
  services/
  store/
  types/
  utils/
```

## Commands

```bash
npm install
npm run build
npm run lint
npm run dev
```
